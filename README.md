# Smart Stadiums & Tournament Operations

A GenAI-enabled solution built for the **PromptWars Challenge 4** (FIFA World Cup 2026). This application enhances stadium operations and the overall tournament experience for fans, organizers, and staff through two dynamic, glassmorphism-designed dashboards.

## Features

- **Fan Dashboard**: Interactive navigation, food/beverage access, accessibility routing, and sustainability insights.
- **Operations Dashboard**: Real-time staff intelligence, mock predictive crowd heatmaps, and AI actionable insights for incident response.
- **GenAI Smart Assistant**: A contextual AI chatbot that provides multilingual assistance to fans and operational intelligence to staff.

## Technology Stack

- React (Vite)
- Recharts (Data visualization)
- Lucide React (Premium iconography)
- Vanilla CSS (Custom Glassmorphism Design System)

## Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the dev server:
   ```bash
   npm run dev
   ```
