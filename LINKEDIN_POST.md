Excited to present **PITCHSIDE**, my submission for the FIFA World Cup 2026 Hackathon (Challenge 4 — Smart Stadiums & Tournament Operations)! 🏟️⚽

PITCHSIDE is a high-performance, real-time command center UI designed to replace scattered dashboards with a single "Operations Intelligence" pane of glass.

**Key Features Built During the Hackathon:**
✨ **Serverless AI Proxy**: Deployed on Vercel, this securely connects a Hugging Face open-weight Llama model directly into the dashboard, without exposing API keys to the client.
✨ **Live Data Integration**: Wired up genuine live weather data via Open-Meteo, proving the end-to-end data pipeline.
✨ **Graceful Fallbacks**: The AI pipeline features a seamless "Demo Mode" fallback, ensuring the interface never looks broken during rate limits or network failures.
✨ **Comprehensive Routing**: Features dedicated modules for Crowd Density, Transportation flow, and Accessibility management, fulfilling the holistic scope of the challenge.
✨ **CI/CD Rigor**: Fully unit-tested with Vitest, wired to GitHub Actions for automated linting and testing on every push.
